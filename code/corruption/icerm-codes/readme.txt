- QTRK_Algorithm.m and mQTRK_Algorithm.m are the implementations of QTRK and mQTRK resp.

- QTRK_trials.m runs QTRK over multiple trials. QTRK_plots.m generates error plots for QTRK.

- QTRK_experiments.m allows the user to input different parameters, run QTRK over multiple trials, and generate the plots.



- mQTRK_QTRK_trials.m runs QTRK and mQTRK (on the same problem) over multiple trials/systems and mQTRK_QTRK_plots.m plots the results.

- mQTRK_QTRK_experiments.m allows the user to input different parameters, run QTRK and mQTRK over multiple trials, and generate the plots.
